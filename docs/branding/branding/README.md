# Branding Pack — Agentic AI Orchestrator

This folder contains logo assets and design tokens for the dashboard and desktop app.

## Files
- `logo-original.png` and `logo-{1024|512|256|128|64}.png`
- `app-icon.ico` and `favicon.ico` (+ 16/32/48 PNGs)
- `tokens.json`: color/radius/elevation tokens
- `../dashboard/src/styles/brand.css`: CSS variables for React/Vite

## Usage
- Web: import `brand.css` once in your app shell and reference CSS variables.
- Desktop (WPF): use `app-icon.ico` for the window/application icon and splash.
