# Star Wars Symbol (Text) — Static Web App

A tiny, licensing-safe static web app that displays the text **“STAR WARS”** (no official logo assets). Includes a small interactive toggle + size slider (stored in `localStorage`).

> Disclaimer: “Star Wars” and related marks are trademarks of their respective owners. This project is a fan-made educational demo and does **not** include any official/copyrighted logo assets.

## Tech stack

- Plain HTML/CSS/JS (no framework)
- Local dev server: [`live-server`](https://www.npmjs.com/package/live-server)
- Build step: copy static files into `dist/`

## Project structure (expected)

```
.
├─ index.html
├─ styles.css
├─ app.js        # JS logic (text-only symbol + controls)
├─ package.json
└─ dist/         # generated by `npm run build`
```

### Important note about script filename
Your `package.json` build script copies `script.js`:

```json
"build": "mkdir -p dist && cp -R index.html styles.css script.js dist/ 2>/dev/null || true"
```

…but the provided app logic file is named `app.js`. Either:

- **Option A (recommended):** rename `app.js` → `script.js`, and ensure `index.html` loads it, **or**
- **Option B:** update the build script to copy `app.js` instead of `script.js`.

(Also ensure `index.html` links to `styles.css` and the JS file you use if you want to rely on the external files rather than inline styles.)

## Prerequisites

- Node.js 18+ (any modern LTS should work)
- npm (comes with Node)

## Install

```bash
npm install
```

## Local development

Starts a local static server and opens the app in your browser:

```bash
npm run dev
```

By default this uses port **5173**:

- URL: http://localhost:5173

You can also use:

```bash
npm start
```

## Build (static output)

Creates a `dist/` folder containing the static site:

```bash
npm run build
```

## Preview the build

Serves the `dist/` output on port **4173**:

```bash
npm run preview
```

- URL: http://localhost:4173

## Static hosting

Because this is a purely static site, you can host the contents of `dist/` on:

- GitHub Pages
- Netlify
- Vercel (static)
- Any basic HTTP server

If you prefer not to use `live-server`, you can also serve locally with e.g.:

```bash
npx serve .
# or after build:
npx serve dist
```

## Accessibility

- The symbol can be activated via click or keyboard (Enter/Space).
- Slider changes are announced via a polite live region.
- Focus-visible styles are included.
