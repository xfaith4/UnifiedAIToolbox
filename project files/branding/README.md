# Portfolio Favicon Collection — Icon Only

Generated from the provided branding source by removing the adjacent “Agentic AI Orchestrator” wordmark/text and centering the icon in a square canvas.

## Recommended files for a web app

Place the files in `web/` at the root of your public/static folder:

```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#061A3A">
```

## Included

- `favicon.ico` — multi-size ICO: 16, 32, 48, 64, 128, 256
- `icons/portfolio-icon-master-1024-transparent.png`
- Transparent PNG sizes: 16, 32, 48, 64, 128, 180, 192, 256, 512

## Branding recommendation

Use this same base icon across the portfolio for recognition. For individual apps, prefer subtle variants rather than totally different logos: accent ring, small badge, or app-specific color treatment. Keep the silhouette identical so the apps feel like one product family.
